# Курсовая работа: База данных HH

## Описание
Проект получает данные о компаниях и вакансиях с HH.ru, сохраняет их в PostgreSQL и предоставляет удобные методы для анализа.

## Установка
1. Клонируйте репозиторий
2. Создайте виртуальное окружение и установите зависимости:
pip install -r requirements.txt
3. Создайте `.env` файл с данными для подключения к PostgreSQL

## Запуск
python src/main.py


## Функционал
- Создание базы данных и таблиц
- Заполнение 10 компаниями и их вакансиями
- Методы:
  - Компании и количество вакансий
  - Все вакансии
  - Средняя зарплата
  - Вакансии с зарплатой выше средней
  - Вакансии по ключевому слову
